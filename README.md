# AI Quiz Application

[English](#english) | [日本語](#日本語)

## English

### Overview
This project is designed to test users' understanding of AI and LLM (Large Language Models) concepts through an interactive quiz application.

### Tech Stack

#### Frontend
- **UI Framework**
  - TailwindCSS
  - shadcn/ui
  - React Bits
- **Animation**
  - Framer Motion
  - React Motion

#### Backend
- Prisma ORM

### Getting Started

1. Clone the repository
```bash
git clone [repository-url]
```

2. Install dependencies
```bash
pnpm install
```

3. Run the development server
```bash
pnpm dev
```

### Project Structure
```
├── app/          # Next.js app directory
├── components/   # React components
├── prisma/       # Database schema and migrations
├── public/       # Static assets
└── lib/          # Utility functions and configurations
```
